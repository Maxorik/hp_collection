*** Парсер html-страницы, содержащей информацию о фигурках ***

Данные берутся с сайта https://brickset.com/minifigs/category-Harry-Potter 

1 - получение изображений
1.1 Выбрать фильтр по фигуркам. Для удобства отфильтровать по году.
1.2 Выбрать вывод 500 фигурок. Перезагрузить страницу для применения фильтра.
1.3 Соранить веб-страницу полностью (нужно для получения изображений фигурок).

2 - постобработка изображений
2.1 убрать фон у изображений с помощью скрипта removebg.py или на сайте https://carve.photos/upload
2.1.1 При необходимости переформатировать картинки в PNG https://www.iloveimg.com/ru/jpg-to-image/jpg-to-png
2.2 сжать изображения с помощью скрипта compress.py или на сайте https://www.iloveimg.com/ru/compress-image
2.3 Полученные изображения переместить в public\catalog_images

3 - получение необработанного стора и парсинг
3.1 возвращаемся на сайт с отфильтрованными фигурками
3.2 инспектируем документ, копируем весь html-dom  в файл brickset.html
3.2 схлопываем раздел <section class="setlist minifiglist"> и копируем его в файл init_data.html
3.3 открываем init_data.html в браузере, запускаем по кнопке START парсер из getter.js
3.4 после появления текста нажимаем COPY и вставляем все в файл src\store\figuresStore.ts, предварительно полностью очистив его.
в getter.js можно указать и настроить желаемый выводимый результат.

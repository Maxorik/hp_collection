*** Парсер html-страницы, содержащей информацию о фигурках ***

Данные берутся с сайта https://brickset.com/minifigs/category-Harry-Potter 

1 - получение изображений
1.1 Выбрать фильтр по фигуркам. Для удобства отфильтровать по году.
1.2 Выбрать вывод 500 фигурок. Перезагрузить страницу для применения фильтра.
1.3 Соранить веб-страницу полностью - в режиме "LIST" (нужно для получения изображений фигурок).
1.4 Полученные изображения скукожить на сайте https://www.iloveimg.com/ru/compress-image (открывать в инкогнито)
1.4.1 При необходимости переформатировать картинки в PNG https://www.iloveimg.com/ru/jpg-to-image/jpg-to-png
1.5 Полученные изображения переместить в public\catalog_images

2 - получение необработанного стора и парсинг
2.1 возвращаемся на сайт с отфильтрованными фигурками
2.2 инспектируем документ, копируем весь html-dom  в файл brickset.html
2.2 схлопываем раздел <section class="setlist minifiglist"> и копируем его в файл init_data.html
2.3 открываем init_data.html в браузере, запускаем по кнопке START парсер из getter.js
2.4 после появления текста нажимаем COPY и вставляем все в файл src\store\catalog_min.js, предварительно полностью очистив его.

в getter.js можно указать и настроить желаемый выводимый результат.
